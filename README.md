# NammaHomeStay

NammaHomeStay is an Android application for managing home stays.

## Prerequisites

To build and run this project, ensure you have the following environment setup:

### SDK Requirements
- **Android SDK:**
    - **Compile SDK Version:** 35
    - **Target SDK Version:** 35
    - **Minimum SDK Version:** 24

### Tooling & Languages
- **Java Development Kit (JDK):** 17
- **Kotlin:** 2.2.10
- **Android Gradle Plugin (AGP):** 9.2.0

## Getting Started

1. Clone the repository.
2. Open the project in **Android Studio (Ladybug | 2024.2.1 or newer)** for best compatibility with AGP 9.2.0.
3. Ensure you have the **google-services.json** file in the `app/` directory (Firebase configuration).
4. Sync the project with Gradle files.
5. Build and Run.
